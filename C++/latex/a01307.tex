\doxysubsection{Detailed Description}
A simple wrapper for real3 that is able to determine whether or not it\textquotesingle{}s defined. 

Upon construction, an optional\+\_\+real\+\_\+3 is put into 2 states, valid or invalid. Constructing this with the empty constructor will create an invalid \mbox{\hyperlink{a01307}{optional\+\_\+real3}}, using the x,y,z parameters will create a valid \mbox{\hyperlink{a01307}{optional\+\_\+real3}}. This can be checked by checking the truth the object similar to std\+::optional (i.\+e. if(optional\+\_\+real3) ).

\begin{DoxyRemark}{Remarks}
This is used in place of std\+::optional, since std\+::optional doesn\textquotesingle{}t support arrays. In many ways it functions similarly, but doesn\textquotesingle{}t feature the exact same syntax for the sake of simplicity. This doesn\textquotesingle{}t use any extra memory than a real3, and comes with the benefit of being able to clearly indicate to callers that a function doesn\textquotesingle{}t always return a usable value and differentiate between code for checking a point and code for accessing a point in a single location. 
\end{DoxyRemark}
