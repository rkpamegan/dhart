get codepoint from 4 hex characters following {\ttfamily \textbackslash{}u} 

For input \char`\"{}\textbackslash{}u c1 c2 c3 c4\char`\"{} the codepoint is\+: (c1 $\ast$ 0x1000) + (c2 $\ast$ 0x0100) + (c3 $\ast$ 0x0010) + c4 = (c1 \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} 12) + (c2 \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} 8) + (c3 \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} 4) + (c4 \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} 0)

The conversion is done by subtracting the offset (0x30, 0x37, and 0x57) between the ASCII value of the character and the desired integer value.
